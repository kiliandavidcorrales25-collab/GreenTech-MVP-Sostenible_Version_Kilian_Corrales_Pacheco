# GreenTech MVP Sostenible - Kilian David Corrales Pacheco

## Resumen 
Este proyecto es una versión optimizada de la landing page **"Salvemos el Planeta"**.  El objetivo es eliminar toda la “grasa digital”: dependencias externas y scripts innecesarios que hacían que la página tardara mucho en cargar y hacerla mas ligera para todo usuario

## Los cambios
- Eliminamos **Bootstrap, jQuery, Moment.js y Font Awesome**.  
- Todo el diseño está echo en un CSS nativo así podemos controlar para que sea mas manejable a nuestros requerimientos
- Organizamos un poco mejor el botón conservando su diseño original pero mas optimo
- Reorganizamos el Footer sin scripts extras que no entendemos

## ¿Con que objetivo hacemos esto?

El objetivo es mejorar la carga de la página para que sea más rápida y consuma menos KB, lo que se significa que es un menor consumo de datos y asi sea posbile en cargar en dispositivos con bajos recursos o móviles,  además, reduce el impacto medioambiental al enviar menos datos, disminuyendo la energía consumida en servidores y redes
